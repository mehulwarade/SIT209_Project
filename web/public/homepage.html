            Web Terminators

Web Terminator is a group of three students (shown below), studying software enginnering at Deakin university. SHROUD is our own home made diy project which comprises of a Raspberry Pi 3 (RPi) powered robot which is controlled over the internet by the user using the web-browser. Click on "Get Started" button above to take control of this robot yourself!

This project was developed under the group project given to us by Kevin Lee (Director of SIT209 - Developing Internet of Things).We have tried our best to make this website and the robot as best as we can, but suggestions are always welcomed.
